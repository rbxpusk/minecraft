# Complete Guide

## Quick Start

1. Run `python main.py`
2. Enter your server IP, port, username, password
3. Click Connect
4. Done!

## Features

### Dashboard
- **Performance Stats**: CPU, RAM, Players, Uptime (auto-updates every 10s)
- **Quick Commands**: 20+ admin commands
- **Console**: Real-time server output
- **Auto-refresh**: Live log updates (5s)
- **Auto-update stats**: Performance stats refresh automatically (10s)
- **Auto-fetch**: Data loads automatically when you switch tabs

### Mods
- **Upload**: Drag and drop or browse
- **Download**: Paste URL from CurseForge/Modrinth
- **Quick Install**: Popular mods with one click
- **Search**: Filter mods instantly
- **Export**: Save your mod list
- **Double-click**: Delete mods quickly

### Players
- **Smart Selection**: Click player, use any button
- **Double-click**: Quick action menu
- **Actions**: Op, deop, kick, ban, message, gamemode
- **Whitelist**: Easy management

### Files
- **Browse**: View all server files
- **Logs**: View and clear
- **Backups**: Create and restore

### Settings
- **Properties**: Edit server.properties
- **Quick Settings**: Common options

## Tips

**Modded Servers**: Use Fabric for best performance

**Memory**: 4GB minimum, 8GB recommended

**Backups**: Before major changes

**Mods**: Check compatibility, install Fabric API first

**Players**: Whitelist for private servers

## Keyboard Shortcuts

- `Enter` in command box: Send command
- `Double-click` mod: Delete
- `Double-click` player: Quick menu

## Troubleshooting

**Can't connect**: Check IP, port, credentials

**Server won't start**: Check Java installed, EULA accepted

**Mods not loading**: Restart server, check compatibility

**Performance issues**: Reduce view distance, add RAM
